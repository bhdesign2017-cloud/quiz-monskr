# MONSKR 主機選購測驗

一個互動式測驗應用程式，幫助使用者找到最適合的 vaping 設備類型。

## 功能特色

- 🎯 **智能推薦系統** - 根據 5 個問題推薦最適合的設備類型
- 🎨 **現代化設計** - 深色主題搭配 MONSKR 品牌黃色
- 📱 **響應式設計** - 完美支援桌面和行動裝置
- ⚡ **快速載入** - 純 HTML/CSS/JS，無需建置步驟

## 設備類型

測驗會根據使用者的偏好推薦以下三種設備之一：

1. **一次性主機** - 適合追求極致便利的使用者
2. **換彈式主機** - 適合喜歡簡單操作的使用者
3. **注油式主機** - 適合追求性能和客製化的使用者

## 技術架構

- **Tailwind CSS** - 透過 CDN 載入
- **Google Fonts** - Noto Sans TC 字體
- **Vanilla JavaScript** - 無框架依賴

## 本地開發

直接在瀏覽器中開啟 `index.html` 即可使用。

## 部署

本專案已部署到 Vercel，可直接訪問線上版本。
